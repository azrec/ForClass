JavaScript 冒泡和捕獲的差異


<div id="parent">
    <button id="child">Click me!</button>
</div>

<script>
    // 在父元素上添加事件監聽（捕獲）
    document.getElementById("parent").addEventListener("click", function() {
        alert("Parent clicked! (capturing)");
    }, true); // 設置為捕獲模式

    // 在子元素上添加事件監聽（冒泡）
    document.getElementById("child").addEventListener("click", function() {
        alert("Child clicked!");
    });
</script>

Git衝突

Git的衝突指的是多個人同時更改同一檔案的同一部分，使得Git無法確定哪個更改才是正確的， 所以Git會在檔案內註記哪邊發生衝突了，方便開發者決定該如何修改。
